# TaskFlow

A clean and responsive **Task Manager Web Application** designed to help users organize daily tasks efficiently and boost productivity.

## 🚀 Live Demo

[Visit Website](https://itsadarshpal.github.io/TaskFlow/)

## ✨ Features

- ➕ Add new tasks  
- ✏️ Edit tasks  
- ❌ Delete tasks  
- 📌 Set priorities  
- 📅 Add due dates  
- ✅ Mark tasks as completed  
- 📊 Track task progress  
- 🔍 Easy-to-use interface  
- 📱 Fully responsive design  

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript
